## Getting Started

### Installing

1. Clone the repo:

```bash
git clone [(https://github.com/raniadeiri/Portifolio.git)]
```
2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
